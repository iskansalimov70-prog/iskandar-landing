# Iskandar — Landing Page

Продающий лендинг для контент-маркетолога / AI-специалиста. Консультации по контенту, стратегии и воронке.

## Стек

Статический сайт: чистые HTML/CSS/JS, без сборки и зависимостей. Открывается напрямую (`index.html`) или деплоится как статика на любой хостинг (GitHub Pages, Netlify, Vercel).

## Структура

```
index.html     — разметка страницы
css/style.css  — стили (дизайн-система: чёрный + красный, Oswald + Onest)
js/script.js   — скролл-анимации появления и аккордеоны (вопросы/FAQ)
```

## Локальный запуск

Просто открыть `index.html` в браузере — сервер не требуется.
